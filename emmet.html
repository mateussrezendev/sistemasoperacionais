Child: >

Abreviação: nav>ul>li

Você pode usar o operador > para aninhar elementos um dentro do outro:

Sibling: +

Abreviação: div+p+b

Use o operador + para colocar os elementos próximos uns dos outros, no mesmo nível:

Climb-up: ^

Abreviação: div+div>p>span^b

Com o operador >, você está descendo na árvore gerada e as posições de todos os elementos irmãos serão resolvidas em relação ao elemento mais profundo:

Grouping: ()

Abreviação: div>(header>ul>li)

Os parênteses são usados ​​pelos usuários avançados do Emmets para agrupar subárvores em abreviações complexas:

Multiplication: *>

Abreviação: ul>li*5

Com o operador *, você pode definir quantas vezes o elemento deve ser exibido:

Text: {}

Abreviação: a{click me}

Você pode usar chaves para adicionar texto ao elemento:

ID: #

Abreviação: #id

Este operador retorna um 'div' com um atributo de ID, 'id':

Class: .

Abreviação: .us

Este operador retorna um 'div' com um atributo CLASS, 'us':
